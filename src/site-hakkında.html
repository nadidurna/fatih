Bu Web Sitesi Neden Var ?

HTML ile basit bir websitesi tasarlayıp, bunu CSS ile şekillendirip, cloud bilgimi de geliştirme amacı ile Amazon S3 statik websitesi servisini kullanıp, bu websitesinin güvenliğini sağlama amacıyla Amazon CloudFront ile HTTPS kullanmayı daha sonrasında kendime Amazon Route53 ile bir domain almayı bunun sonrasında ise javascript ile özgeçmiş sayfası için bir ziyaretçi sayfası oluşturmayı, ziyaretçi sayısını gösterebilmek ve güncelleyebilmek adına DynamoDB servisini kurmayı, sonrasında web uygulamasından gelen istekleri kabul eden veritabanı ile iletişime geçen bir API yaratarak bulut servisinin API Geçidini kullanmayı, CI/CD kullanarak test ortamlarından geçtikten sonra bulut servisinin sağladığı uygulama üzerinden ayağa kaldırmayı, son olarak da bir blog yazısı açıp hem bu proje üzerinde çalışırken öğrendiğim şeyleri yazıp hem de halihazırda çözmüş ve çözmekte olduğum CTF’ lerin çözümünü buraya dahil etmeyi planlamaktayım.
